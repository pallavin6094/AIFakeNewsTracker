# AIFakeNewsTracker

## Overview
AIFakeNewsTracker is a Java-based application that helps identify and track fake news articles and social media posts. The system analyzes content credibility, source reliability, and viral risk to assist users in detecting potentially misleading information.

## Features
- Add and manage news articles
- Track social media posts
- Analyze content credibility
- Evaluate source reliability
- Calculate viral risk scores
- Store analysis results in a database
- Generate reports on fake news trends

## Technologies Used
- Java
- JDBC
- MySQL
- Object-Oriented Programming (OOP)
- SQL

## Project Modules
1. News Article Management
2. Social Media Post Tracking
3. Credibility Analysis
4. Source Reliability Assessment
5. Viral Risk Detection
6. Report Generation

## Database
The project uses MySQL for storing:
- Articles
- Posts
- Sources
- Credibility Analysis Results
- User Information

## How to Run
1. Clone the repository.
2. Create the MySQL database.
3. Import the provided SQL script.
4. Update database credentials in the project.
5. Compile and run the application.
